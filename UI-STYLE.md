# Perihelion Remote Control UI 风格指南

## 设计哲学

界面是工具，不是装饰。以冷白为基底，用精确的边框和微妙的背景色差构建层级，拒绝一切多余的视觉噪声。品牌橙在克制的灰白体系中成为唯一的视觉锚点，像监控面板上的告警灯——平时安静，出现即有意义。

## 关键词

精确 · 克制 · 冷静 · 功能 · 硬朗

## 字体系统

- **UI 字体**：`"Geist", -apple-system, "Segoe UI", sans-serif`
- **代码字体**：`"Geist Mono", "Courier New", monospace`

字重使用规则：
- 大标题 / 品牌名：`700 / Bold`
- 小标题 / 强调标签：`600 / SemiBold`
- 正文 / 列表项：`400 / Regular`
- 辅助说明 / muted：`400`（用颜色而非字重降权）

字号层级：`11 / 12 / 13 / 14 / 16 / 18`（单位 px，UI 统一 14px 基准）

## 视觉风格

**圆角**：整体偏锐利
- 按钮、输入框：`6px`
- 卡片、弹窗：`8px`
- 消息气泡：`12px`（保留方向感）
- 小标签/chip：`4px`

**阴影**：**完全不使用阴影**。层级全部靠以下手段：
1. 背景色差（`#FFFFFF` vs `#FAFAFA` vs `#F4F4F5`）
2. `1px solid` 边框线
3. 左侧竖线作为视觉锚点（选中态、强调块）

**图标风格**：线性图标，stroke-width 1.5，Lucide 图标库

**动效节奏**：快速克制
- 过渡时长：`120ms ~ 150ms`，`ease` 曲线
- 消息出现：`fadeInUp 0.15s`（`translateY(4px)` → `0`）
- 不使用弹性/弹跳动效

## 组件风格要点

**按钮（主要）**：
- 背景 `#EA580C`，文字 `#FFFFFF`，圆角 `6px`，无阴影
- hover：背景换 `#C2410C`，`transition 0.12s`
- active：`scale(0.97)`

**按钮（次要/幽灵）**：
- 背景透明，边框 `1px solid #E4E4E7`，文字 `#3F3F46`
- hover：背景 `#F4F4F5`

**输入框**：
- 背景 `#FFFFFF`，边框 `1px solid #E4E4E7`，圆角 `6px`
- focus：边框换 `#EA580C`，`box-shadow: 0 0 0 3px rgba(234,88,12,0.08)`（仅 focus ring）
- placeholder：`#A1A1AA`，不加斜体

**侧边栏**：
- 背景 `#FAFAFA`，右边框 `1px solid #E4E4E7`
- 选中项：背景 `#FFFFFF`，左侧 `2px solid #EA580C` 竖线
- hover：背景 `#F4F4F5`

**消息气泡（用户）**：
- 背景 `#EA580C`，文字白，圆角 `12px 12px 3px 12px`，无阴影

**消息气泡（助手）**：
- 背景 `#FFFFFF`，边框 `1px solid #E4E4E7`，圆角 `3px 12px 12px 12px`，无阴影

**工具卡片**：
- 背景 `#FFFFFF`，边框 `1px solid #E4E4E7`，圆角 `6px`
- 头部背景 `#FAFAFA`，底部边框 `1px solid #F0F0F1`
- 工具名颜色 `#16A34A`，字体 Geist Mono

**弹窗**：
- 背景 `#FFFFFF`，边框 `1px solid #E4E4E7`，圆角 `10px`，无阴影
- 遮罩层：`rgba(9,9,11,0.4)` + `backdrop-filter: blur(4px)`

**HITL 审批项**：
- 背景 `#F4F4F5`，左侧 `2px solid #16A34A`，圆角 `6px`

**加载点**：
- 颜色 `#EA580C`，垂直弹跳动画

**在线状态点**：
- 在线：`#16A34A`，带 `online-ring` 脉冲动画
- 离线：`#E4E4E7`

## 参考产品

- **Linear**：整体网格感、精确边框、冷静色调
- **Vercel Dashboard**：zinc 色系的白灰层级、文字权重对比
- **Zed Editor**：Geist 字体的运用方式
- **Railway**：无阴影纯边框卡片系统

## 刻意回避

- ❌ 任何 `box-shadow`（包括卡片投影、按钮投影、悬浮阴影）
- ❌ 暖色系底色（米色、奶油色、纸张色）
- ❌ 渐变背景、磨砂玻璃（blur 仅用于弹窗遮罩）
- ❌ 圆润的大圆角（> 12px）
- ❌ 装饰性插图、emoji、彩色标签
- ❌ 多于两个 accent 色同时出现
- ❌ Inter、Roboto 等泛用字体
